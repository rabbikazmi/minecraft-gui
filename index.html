Minecraft GUI Inventory System

A fully interactive Minecraft-style inventory crafting system built with HTML5, CSS3, and JavaScript.

Features:

  • Advanced click-based inventory management
  • 2x2 crafting grid with authentic recipes
  • Character selection with speech bubbles
  • Interactive recipe book
  • Day/Night theme switching
  • Sound effects with Tone.js
  • Responsive design

Recipes Available:

Bread, Cake, Maps, Firework Rockets, Tools, Furniture, and more!
